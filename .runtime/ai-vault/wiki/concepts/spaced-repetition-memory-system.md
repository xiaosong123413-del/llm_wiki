# [[Spaced Repetition]] 记忆系统

[[Spaced Repetition]] 是一种基于以下原理的记忆优化方法：在策略性递增的间隔时间内复习材料，比集中练习能产生更强的长期记忆保持效果。它通过在学习者可能遗忘材料之前的关键时刻安排每次复习，直接应对遗忘曲线。

## 核心机制

该系统通过在初次学习后逐渐延长复习间隔来工作。典型的五阶段时间表如下：

- 复习1：初次学习当天晚些时候
- 复习2：第二天
- 复习3：初次学习后四到七天
- 复习4：初次学习后八到十四天
- 复习5：通过文章和真实文本等语境材料重新接触

每次成功回忆都会延长下一个间隔；困难或失败则会缩短间隔。这种自适应调度将精力集中在薄弱项目上，同时减少在已掌握项目上花费的时间。

## 与遗忘曲线的关系

[[Spaced Repetition]] 是对记忆随时间自然衰减的直接对策。没有刻意复习，新学习的材料会快速淡忘。通过将每次复习时间安排在记忆保持开始下降的时刻，该系统在每次重复时重建和强化记忆痕迹，逐渐延长信息保持的时间。这使其在大量记忆任务（如词汇习得和法律条文记忆）中特别有效。

## 实施工具

实践中使用三种主要工具来实施[[Spaced Repetition]]：

- [[Anki]] — 带有内置[[Spaced Repetition]]算法的闪卡软件，根据回忆表现自动安排复习
- 带有DataviewJS插件的[[Obsidian生产力系统]] — 允许学习者在个人知识库内构建自定义复习计划
- 纸质闪卡 — 当数字工具不可用或不理想时仍然有效的低技术替代方案

## 在词汇学习中的应用

一个有记录的使用案例是考试准备词汇学习，其中[[Spaced Repetition]]应用于多周计划的词汇表。初次学习后进行当天复习，然后是每日和每周间隔，最后在期刊文章等真实阅读材料中再次遇到这些词汇。这种最终的语境重新接触强化了超越孤立回忆的意义理解。

## 与更广泛学习管理的整合

[[Spaced Repetition]]解决了[[个人学习管理]]的记忆保持层面问题，但与解决其他两个核心学习问题的方案结合使用效果最佳：管理任务启动时的认知疲劳，以及维持长期学习计划。如果学习者正确安排了复习但由于困倦或拖延无法维持每日学习，就无法充分实现该系统的益处。将[[Spaced Repetition]]与一致的睡眠时间表、任务分解和结构化长期计划相结合，比单独使用能产生更可靠的结果。
