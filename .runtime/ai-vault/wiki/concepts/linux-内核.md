# Linux 内核

## 概述

**Linux 内核** 是 Linux 系统的核心部分，负责进程、内存、设备和文件系统等基础能力。当前知识库主要用它来说明 Android 并非完全独立的技术体系，而是建立在 Linux 内核之上的移动系统。

## 在当前知识库中的位置

围绕 Android 的页面中，Linux 内核承担“底层解释框架”的角色：它说明为什么 Android 具备类似 Linux 的底层结构，同时也解释了应用层为何仍受到更强的沙盒和权限约束。
