# 小红书内容导入器 (Xiaohongshu Importer)

小红书内容导入器是一个专为 Obsidian 设计的社区插件，允许用户将小红书平台上的笔记内容无缝导入到个人知识库中。该插件支持提取笔记的标题、正文、图片、视频和标签信息，并将其转换为结构化的 Markdown 文件，便于在 Obsidian 中进行知识管理和组织。

## 核心功能

### 内容导入能力

小红书内容导入器支持通过分享链接或分享文本导入笔记，兼容多种 URL 格式，包括传统的 `discovery/item` 格式和较新的 `explore` 格式，插件会自动将这些链接标准化处理。导入过程中可以完整提取笔记的标题、内容、图片、视频和标签等元素。

### 分类管理系统

插件提供灵活的分类管理功能，用户可以创建自定义分类（如"旅行"、"美食"等）来组织导入的笔记，未分类的内容会自动归入"其他"类别。分类系统支持添加、编辑、删除和重新排序操作，帮助用户建立个性化的知识结构。

### 媒体处理选项

用户可以选择将图片和视频下载到本地知识库中，或者保持使用原始 URL 进行嵌入。本地下载的媒体文件会统一存储在指定的媒体文件夹中，确保知识库的完整性和离线可用性。

## 技术规格

### 版本信息

- **当前版本**: 1.1.3
- **开发者**: bnchiang96
- **许可证**: MIT
- **代码仓库**: https://github.com/bnchiang96/xiaohongshu-importer

### 安装方式

插件现已在 Obsidian 社区插件市场中提供，用户可以通过 **设置 > 社区插件 > 浏览** 搜索"Xiaohongshu Importer"进行安装。也支持手动安装方式，需要下载发布的 ZIP 文件并解压到插件目录中。

### 文件结构规范

导入的笔记按照 `<默认文件夹>/<分类>/<笔记标题>.md` 的格式保存，媒体文件存储在 `<默认文件夹>/media/` 目录下。每个笔记文件包含标准的 frontmatter 元数据，记录标题、来源链接、日期、导入时间和分类信息。

## 使用限制

### 平台兼容性问题

根据实际使用反馈，该插件在安卓平台上存在兼容性问题，无法正常工作。这一限制影响了移动端用户的使用体验，特别是对于依赖手机进行内容收集的用户群体。

### 网络依赖性

插件的正常运行需要稳定的网络连接来访问小红书内容和下载媒体文件。网络问题或小红书平台的访问限制可能导致导入失败，用户需要确保目标笔记为公开状态且可正常访问。

## 开发环境

插件基于现代 Web 技术栈开发，要求 Node.js 18.x 或更高版本，使用 Yarn 进行依赖管理。开发者可以通过 `yarn dev` 启动开发模式，或使用 `yarn build` 构建生产版本。插件遵循 Obsidian API 规范和社区插件开发指南。
