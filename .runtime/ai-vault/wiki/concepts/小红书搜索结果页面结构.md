# 小红书搜索结果页面结构

小红书（Xiaohongshu / RED）的搜索结果页面是用户通过关键词检索内容的核心入口。页面由多个功能区域组成，包括内容类型筛选标签、过滤器、瀑布流卡片列表以及相关搜索推荐词等模块。

## 内容类型标签栏

页面顶部设有横向标签栏，用于按内容类型筛选结果，已知标签包括：

- 全部
- 图文
- 视频
- 用户

默认展示"全部"类型的混合结果，用户可点击切换至单一内容类型。

## 筛选器

标签栏旁或下方提供"筛选"入口，允许用户对搜索结果进行进一步过滤。具体筛选维度（如时间、地区、互动量等）在源材料中未作详细说明。

## 内容卡片瀑布流

搜索结果主体区域以瀑布流（Masonry Grid）形式呈现内容卡片，每张卡片包含：

- 封面缩略图（图文或视频预览）
- 互动数据（如点赞数，源材料中可见数值如 82、81 等）
- 笔记标题或摘要文字
- 指向具体笔记详情页的链接（URL 格式为 `https://www.xiaohongshu.com/search_result/{note_id}`，附带 `xsec_token` 和 `xsec_source=pc_search` 参数）

## 相关搜索推荐词

页面中穿插展示与当前关键词相关的搜索推荐词条，例如在"南大法学学硕真题"相关搜索下可见：

- 25年南大法学学硕真题
- 南大研究生考试排名
- 南大学硕排名
- 南大临床学硕名单
- 浙江财经大学法学考研用书
- 南大法学学硕题
- 南大学硕题目
- 南大材料专硕考试科目

这些推荐词以文字链接形式嵌入瀑布流之间，引导用户进行关联搜索。

## 页面交互元素

- "回到顶部"按钮：页面滚动后出现，点击可快速返回顶部
- "加载中"状态提示：瀑布流底部在异步加载更多内容时显示，表明页面采用无限滚动（Infinite Scroll）加载机制

## URL 结构说明

笔记详情页链接遵循以下模式：

```
https://www.xiaohongshu.com/search_result/{note_id}?xsec_token={token}&xsec_source=pc_search
```

`xsec_source=pc_search` 参数表明该链接来源于 PC 端搜索结果页，可用于来源追踪与分析。
