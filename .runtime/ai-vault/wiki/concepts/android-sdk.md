# Android SDK

## 概述

**Android SDK** 是 Android 应用开发所需的软件开发工具包。在当前知识库材料中，它被描述为编译、调试和运行 Android App 所需库与工具的集合。

## 与 Android Studio 的关系

现有原料明确指出，[[Android Studio]] 会把 Android SDK 作为开发环境中的核心组成部分一并安装或配置。对大多数入门开发流程来说，SDK 不是独立于 IDE 的额外概念，而是开发环境本体的一部分。

## 在开发流程中的位置

围绕当前知识库的 Android 入门材料，Android SDK 主要承担三类职责：

- 提供编译与构建所需的库和工具
- 支撑调试与运行 Android 应用
- 与虚拟设备、构建系统一起组成基础工具链
