# JSON Schema

## 概述

**JSON Schema** 是一种描述 JSON 数据结构与字段约束的规范。当前知识库里，它主要出现在 AI 工具调用场景，用来定义 `parameters` 这类输入对象的字段类型、必填关系和格式要求。

## 在当前知识库中的作用

相关页面把它当作工具调用参数约束的标准，而不是孤立的数据格式知识点。它和 AI 模型生成结构化参数、以及开发者约束工具输入的场景直接相关。
