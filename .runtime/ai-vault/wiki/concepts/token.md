# token

## 概述

在当前知识库里，**token** 主要被用来表示语言模型处理文本时的上下文预算单位。对 AI 编码工具而言，命令输出、日志噪声和重复信息都会占用 token，并直接影响成本与有效上下文容量。

## 在当前材料中的作用

相关页面集中讨论了两类问题：一类是 token 被无效输出浪费，另一类是上下文过长导致模型压缩历史或遗忘细节。因此 token 既是成本概念，也是上下文管理概念。
