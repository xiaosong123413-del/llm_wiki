# SmartClip

## 概述

**SmartClip** 是当前知识库里反复出现的一款网页剪藏工具。它的核心价值不只是手动保存网页，而是通过 MCP 服务和配套 Skills 把网页抓取能力接入 AI Agent 工作流。

## 在当前知识库中的作用

相关页面把 SmartClip 视为“可被 Agent 调用的剪藏基础设施”。它既可以作为浏览器插件单独使用，也可以通过 `smartclip-mcp`、`smartclip-read`、`smartclip-save` 等能力进入自动化流程。
