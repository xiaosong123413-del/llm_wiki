# Hermes Agent

## 概述

在当前知识库的工具分层里，**Hermes Agent** 被放在“终端优先的本地机器智能体”位置。它主要承担命令行操作、本地环境执行和系统级任务。

## 在工具链中的角色

现有页面通常把 Hermes Agent 视为执行层组件，而不是知识组织层组件。它更接近本地运行时和操作接口，适合承接需要直接接触文件系统或终端的工作。
