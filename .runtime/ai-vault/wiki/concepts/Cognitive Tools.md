# Cognitive Tools

Cognitive Tools 指帮助人完成理解、比较、记忆和判断的外部工具与抽象方法。它既可以是 [[Mental Models]] 这样的思维框架，也可以是 [[Second Brain]] 这样的知识外化系统。

## 在当前知识库中的位置

当前知识库里，这个概念主要用于把“模型”“笔记系统”“决策框架”放进同一个视角下理解：它们共同作用于认知负荷管理，而不是孤立的技巧清单。
