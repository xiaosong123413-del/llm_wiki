# Linux

## 概述

**Linux** 是一类广泛使用的类 Unix 操作系统家族。当前知识库提到它时，重点通常不是桌面发行版本身，而是它作为 Android 底层技术背景和开发语境的一部分。

## 在当前知识库中的角色

Android 相关页面里，Linux 常被用来解释系统底层结构：Android 建立在 Linux 内核之上，但应用层并不直接等同于传统 Linux 桌面环境。因此，它在这里更多是“底层基础”而不是“直接操作对象”。
