# Surge

## 概述

在当前知识库中，**Surge** 主要被当作支持进程级或 Bundle ID 级分流的代理工具使用，特别适合处理桌面主程序与辅助进程需要同时走代理的场景。

## 在当前材料中的角色

相关原料提到，如果某些桌面应用在主程序已经走代理的情况下仍然无法完成授权回调，往往是因为辅助进程没有被纳入代理规则。Surge 这类工具可以通过进程名或 Bundle ID 精确覆盖这些流量。
