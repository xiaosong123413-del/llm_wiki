# Cursor

## 概述

在当前知识库中，**Cursor** 被归入 AI 编码客户端一类，并被明确描述为支持规则与 skills 的工作环境。

## 在当前材料中的角色

相关原料主要记录了它如何接入 skills：既可以从应用市场安装，也可以通过远程规则方式拉取配置。换句话说，当前知识库关注的不是 Cursor 的全部产品能力，而是它作为“支持技能包与规则系统的客户端”这一点。

## 与技能体系的关系

从现有材料看，Cursor 能够与 [[AGENTS.md 配置文件]]、skills 目录和远程规则一起工作，因此它在知识库中的位置更接近“可承载 Agent 规则与技能的宿主环境”。
