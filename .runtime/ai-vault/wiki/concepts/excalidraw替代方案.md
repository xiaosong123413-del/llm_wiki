# Excalidraw替代方案

在 [[Obsidian]] 生态中，白板与手绘图表功能主要由两种方式提供：内置的 Canvas（白板）功能，以及第三方的 Excalidraw 插件。当这两者存在局限时（例如内嵌白板只能显示缩微图而无法展示完整内容），用户往往需要寻找替代方案。

## 背景：内嵌白板的局限

Obsidian 原生白板（Canvas）嵌入到 Markdown 笔记后，仅能显示白板的缩微图，无法在笔记内直接渲染完整的白板内容。这一限制促使部分用户转向 Excalidraw 插件，因为 Excalidraw 支持在 Markdown 文件中内嵌并完整显示绘图内容。

## 主要替代方案

### Excalidraw 插件

Excalidraw 是目前 Obsidian 社区中最常被推荐的白板替代工具。相较于原生 Canvas，它的核心优势在于：

- 支持手绘风格的图形与文字
- 可将绘图内嵌到 Markdown 笔记并完整显示内容，而非仅显示缩微图
- 与 Obsidian 双链系统深度集成

当用户遇到"内嵌白板只显示缩微图"的问题时，社区的直接建议即是尝试 Excalidraw 插件作为替代。

## 选型建议

| 场景 | 推荐方案 |
|------|----------|
| 需要在笔记中完整内嵌并显示绘图 | Excalidraw 插件 |
| 需要卡片式知识组织与节点连接 | Obsidian Canvas（原生白板） |

选择时需优先验证插件的版本兼容性与长期稳定性，这与 [[Agent集成工作流]] 中对工具选型的建议一致——稳定性优先于功能丰富度。
