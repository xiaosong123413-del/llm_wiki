# 文档随机漫游 (Document Random Roam)

文档随机漫游是一种知识复习机制，通过随机打开知识库或云文档中的某一篇文档，触发对已有内容的重新审视，进而帮助用户发现文档间的潜在关联、完善知识图谱结构。

## 功能说明

在[[飞书文档助手]]（`feishu-doc-helper`）中，文档随机漫游作为核心功能之一被明确列出：随机打开一篇飞书云文档，用于复习云文档中的内容，可以进一步总结整理文档之间的关系，构建自己的知识图谱。

## 使用场景

该功能的典型使用场景是知识沉淀后的定期复习。用户在飞书云文档中积累了大量内容后，往往难以主动回顾所有文档；随机漫游通过"随机抽取"的方式打破主动检索的局限，将被遗忘或搁置的文档重新带入视野。在此基础上，用户可以：

- 对文档内容进行二次整理和补充
- 识别与其他文档的关联，手动或借助工具建立双链
- 逐步构建个人知识图谱

## 所属工具

文档随机漫游功能由开源浏览器插件 **飞书文档助手**（`sancijun/feishu-doc-helper`）提供，该插件同时支持批量导出飞书云文档和飞书知识库。插件以 Chrome 扩展形式安装，需手动加载已解压的扩展程序包。

## 与知识管理工作流的关系

文档随机漫游可以作为[[Agent集成工作流]]的补充手段。在自动化程度较低的场景下，随机漫游提供了一种低成本的人工触发式复习路径，与 Agent 驱动的结构维护形成互补——前者依赖人的主动判断，后者依赖规则和自动化。
